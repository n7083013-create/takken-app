## TestFlight 内部テスト(本審査前に必須)

App Store 提出前に TestFlight で動作確認:

1. EAS Build 完了後、`.ipa` がダウンロード可能になる
2. App Store Connect → アプリ → **TestFlight** タブ
3. iOS Build がアップロードされる(EAS が自動 submit する場合あり)
4. **「内部テスト」** グループに自分を追加
5. iPhone で TestFlight アプリをインストール
6. 招待リンクから takken-app をインストール
7. 全機能を動作確認(課金・通知・同期・弱点ドリル)
8. 問題なければ → App Store 提出ボタン
